About the John Peabody Harrington Papers

This collection provides on-campus access to the PDF scans of microfilms from John Peabody Harrington Papers. The Tutt Library also has this collection in microfilm.

Note:

Much of the content in this collection are PDFs of scanned handwritten manuscripts, full-text searching is NOT available for this collection.

Initial cataloguing of this collection was done by Ravi Donepudi. Collection updated to Bootstrap 3 by Ryan Keenan.
